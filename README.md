# Towards Smarter Project Management Tools - User Interface

To clone the project, run the following command:

```bash
git clone https://eng-git.canterbury.ac.nz/fgi18/seng402-smarter-ui.git
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

### View server

Navigate to http://localhost:5173

### Select Assessment Tool

Navigate to http://localhost:5173/change

Click the 'switch' button until you get the tool you want to use; 0 is the NLP tool, 1 is the machine learning tool
